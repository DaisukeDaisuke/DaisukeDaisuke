## 私は、筆記がまったくできません。

事情は伏せますが、私は筆記することがまったくできません。  
一般社会では、この症状は学習障害(LD)の一つでディスグラフィアと呼ばれています。
- ディスグラフィアは、本人が筆記するために書き方を勉強しても、一定期間たつと記憶と同様に忘れる、思い出せない、定着しないという特性のため、私は日常生活において筆記はほとんど捨てました。
    - このため、2020年頃には少しだけ書けた漢字も、ほとんど忘れています。ただし下記の通り、読み方は覚えており、文章構築能力もあるため、自分の強みとして活動に生かしています。
    - 一般の人ができるであろう、「待」の漢字を想像してくださいなどの課題が私にはできません。
        - なので、漢字の読み方は、下記の通り漢字の形を経由しない曖昧な世界を生きています。
        - また、漢字の形は通常記憶と同じ扱いであるため、(そもそも書けないけど)暗記しなければならない場面では漢字の書き方ごと覚えます。
    - 下記でも間接的に述べてますが、私はランダムな思いついた言葉を言う語彙力を推定するテストも苦手です。普通の人換算だと、ほとんどの語彙はhddに保存されていて、必要になった時にhddからフェッチされ、通常時はアクセス拒否される感じだと思います。
- このため、自分の名前以外はすべてひらがなで書き、手続きは代筆してもらい、筆箱とあいうえお表を常に持ち歩くという生活をしています。
- また、筆記で漢字を書くことへの恐怖も多少ながらあるため、2026年時点では書ける漢字でも絶対にひらがなで書きます
    - 恐怖とは、不器用や、脳の障害が原因で漢字の複雑な構造を正しく模写できず、ミスしてしまうのではないかと感じること。
        - 2020年に長文の手紙を筆記することを強要され、苦労したトラウマであるのも上記恐怖の要因でもあります。

日常で困らない程度の文章構築能力があるため、すべての活動はキーボード、マウス、wqhdディスプレイを介して行っています。
- 漢字の形や、思考チェーンに関係ない語彙の引き出しに異常があると思われます。
- 常用漢字程度であればかなりの速度で意味を理解することができます。
    - 意味を理解するという通り、形状読み、グループ読み、文字予測という特殊な読み方をするため、同じ速度で正しく文章を音読できるかについては、「できません」。
    - グループ読みとは、似たような意味の漢字を思い切って1つのグループの代表の意味としてとらえる読み方です。
    - 形状読みとは、脳の自動化チェーンの一つであり、常用漢字の並びのパターンをもとに読むという読み方です。
    - どうでもいい/興味ない文章の場合、形状読みを使用し爆速で読むので、だいたいの意味しか理解してなかったりします。また、複数回読むことも結構あります。ただしこの場合短期記憶には乗りません
    - 漢字の形をイメージできないため、漢字の並びのパターンとその読み方は無意識のうちに丸暗記してます
    - 読書はしません。主にネットニュース(稀)や技術文書などを主に読んでおり、長文だとワーキングメモリからあふれて要約しか残らないし、基礎知識に収録されてなくて読めない、読み方は知らないけどグループ読みの対象で知ってるとか、初見で形状読みが拒否するなどの字とかの場合もあるため、私には読書に向いていません。他にも多分物語に対する理解の能力があまりないからだと思います。
    - ドラマもみません。ただしブラッシュアップライフは面白かったです。
- また、パソコン使用時のフォントはメイリオじゃないと文字がぼやけて見えるため、私は広告ブロッカー、devtoolsを使ってでもメイリオに変更するということをします。

## 書き方の方針

- READMEは短くすること自体を目的にしない。
- 重要なのは、情報の出し順を制御すること。
- L1は名刺、L2は活動説明、L3は公開できる自己説明アーカイブとする。
- 同じ話題をL2とL3の両方で扱ってよい。
- L2では短く、L3では詳しく書く。
- 「エンジニア」「開発者」のような肩書きより、具体的な活動を書く。
- 強い拒否や警告は、必要な場所に置く。ただし、L1では最小限にする。
- SNSで言いすぎそうな内容は、L3に移してREADMEを引用できるようにする。
